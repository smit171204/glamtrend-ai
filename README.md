# 👗 GlamTrend

### Women's clothing Trend Analysis 

------------------------------------------------------------------------

## 📌 Project Overview

GlamTrend is a data-driven fashion analytics system that analyzes
women's clothing reviews to identify trends, perform sentiment analysis,
generate insights, recommend items, and predict future trends using
Machine Learning.

------------------------------------------------------------------------

## 🚀 Features

-   📊 Analytics Dashboard
-   🤖 AI Insights (Sentiment Analysis)
-   🎯 Recommendation System (Age-based)
-   🔥 Trend Detection
-   📈 Trend Forecasting (ML-based)
-   🧠 Smart Chatbot

------------------------------------------------------------------------
